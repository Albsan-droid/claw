# pkg/agent/voice_prompt.go

## 対象ソース
- `pkg/agent/voice_prompt.go`

## 概要
音声入力時にだけシステムプロンプトへ付加する補助文を返す。TTS 読み上げを前提に、短く自然な応答、Markdown 非使用、識別子の逐次読みなどを英語で指示する。

## 責務
- 音声モード専用の固定ガイダンス文字列を返す
- TTS に不向きな表現を避けるよう LLM に指示する

## 主要な型・関数・メソッド
- `voiceModePrompt() string`

## 詳細動作
- 実装は引数無し・分岐無しで、複数行の raw string literal をそのまま返す。
- 文面は英語で固定されており、主な指示は次の通り。
  - 通常は 1〜3 文で短く返す
  - Markdown を使わない
  - 自然な話し言葉にする
  - 数字・英数字識別子・型番・コードは 1 文字ずつ読ませる
  - emoji / emoticon / kaomoji を使わない
  - 詳細要求時は長くしてよいが、自然言語のままにする
  - コードやファイル内容のような技術的出力は text mode を勧める
- `ContextBuilder.BuildMessages` が `inputMode == "voice" || inputMode == "assistant"` のときだけ連結する。

## 入出力・副作用・永続化
### 入力
- なし

### 出力
- 音声モード用の固定プロンプト文字列

### 副作用
- なし

### 永続化
- なし

## 依存関係
- なし（標準ライブラリも未使用）

## エラーハンドリング・制約
- ロケール切替やテンプレート差し替えは実装されていない。
- 返す文面はすべて固定で、入力モード以外の条件分岐はない。
