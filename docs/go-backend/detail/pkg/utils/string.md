# pkg/utils/string.go

## 対象ソース
- `pkg/utils/string.go`

## 概要
Unicode を考慮した単純な文字列切り詰め関数を提供する。

## 責務
- rune 単位での切り詰め
- 必要時のみ `...` を付けた短縮表示の生成

## 主要な型・関数・メソッド
- `Truncate(s string, maxLen int) string`

## 詳細動作
- 文字列を `[]rune` に変換し、バイト長ではなく文字数ベースで扱う。
- `len(runes) <= maxLen` なら元の文字列をそのまま返す。
- `maxLen <= 3` の場合は `...` を付けず、先頭 `maxLen` rune だけ返す。
- それ以外で切り詰めが必要な場合は、先頭 `maxLen-3` rune に `...` を付けて返す。

## 入出力・副作用・永続化
### 入力
- 対象文字列 `s`
- 最大文字数 `maxLen`

### 出力
- 切り詰め後の文字列

### 副作用
- なし

### 永続化
- なし

## 依存関係
- なし（標準ライブラリ import も不要）

## エラーハンドリング・制約
- `maxLen` が負値でも境界チェックはなく、`runes[:maxLen]` で panic しうる実装である。
- 文字数判定は rune 単位であり、表示幅（全角 / 半角幅）までは考慮しない。
